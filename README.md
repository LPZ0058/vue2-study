**测试**

选择[mocha](https://mochajs.org/)作为测试框架
选择chai作为断言库


**Git 提交规范**

选择commitlint对Git的提交信息进行校验（要符合约定式提交）
选择commitizen注入提交信息，并用cz-customizable自定义git cz的注入内容

**日志变更**

选择Standard Version去自动变更日志（挂到 post-commit钩子上）


**检查测试覆盖率**

选择Istabul检查代码覆盖率


**构建工具**

选择webpack作为构建工具
